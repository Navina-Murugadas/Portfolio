My works

BANK LOAN ANALYSIS

This project delves into detailed analysis of bank loan data, examining key metrics such as loan applications, funded amounts, interest rates, and debt-to-income ratios. Leveraging SQL for robust data extraction and Power BI for insightful visualization, it aims to offer comprehensive insights into loan dynamics and quality assessment.

LAPTOP PRICE PREDICTOR

The Laptop Price Predictor project is designed to forecast laptop prices, saving users both time and money. By using this tool, you can avoid the hassle of visiting multiple stores to compare prices, making the process of purchasing a new laptop more efficient and convenient.

GOOGLE PLAYSTORE APPS ANALYSIS

This project analyzes the Google Play Store's vast ecosystem, providing insights into app categories, user reviews, ratings, and more. By collecting and cleaning data, it aims to offer valuable information for developers, businesses, researchers, and users about the mobile app marketplace.

CREDIT CARD FRAUD DETECTION

This project utilizes a Decision Tree Classifier to detect credit card fraud due to its interpretability and effectiveness in capturing non-linear relationships. It focuses on enhancing transaction security by accurately identifying fraudulent activities.

MOVIE RECOMMENDER SYSTEM

In this project, a Movie Recommender System is developed using cosine similarity and content-based techniques. It aims to enhance user experience on streaming platforms by offering personalized movie recommendations tailored to individual viewing preferences.

EMAIL SPAM CLASSIFIER

This project uses Multinomial Naive Bayes to classify emails as spam or not spam, helping to keep your inbox free from unwanted messages. It aims to enhance email security by identifying and filtering out spam efficiently.
